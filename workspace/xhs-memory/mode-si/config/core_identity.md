# 模式一：成长小磁星 (Super Individual)
## 核心设定档案

---

## 1. 基础定位

### 账号身份
- **中文名**: 超级个体进化室
- **英文名/缩写**: SI (Super Individual)
- **Slogan**: "个体帮助个体，互动交流，思想碰撞"
- **副标**: "每周更新个人成长的思考，激发灵感"

### 人设定位
**商业认知布道者** —— 通过挖掘成功人士早期经历，提炼可复制的成长路径

### 目标人群
- 25-35岁职场进阶人群
- 渴望财富自由的创业者
- 对商业思维感兴趣的自我提升者

---

## 2. 内容体系

### 核心选题方向
| 类型 | 占比 | 说明 |
|-----|------|------|
| **大佬早期考古** | 40% | 张一鸣/黄峥/巴菲特早期博客、演讲、访谈 |
| **方法论提炼** | 30% | 商业框架、思维模型、认知升级 |
| **编年史梳理** | 20% | 人物成长时间线、关键节点解读 |
| **海外内容汉化** | 10% | Naval Ravikant、Dan Koe等思想领袖 |

### 标题公式库
```
公式A: [名人] + [早期/刚毕业] + [核心洞察]
  → "张一鸣刚毕业时就已经开悟"
  → "王兴8年前的讲话就已经预判到"

公式B: [名人] + [堪称一绝/一流] + [方法论]
  → "美团王慧文：堪称一绝的经营分析框架"
  → "段永平浙大访谈：最重要的不是勤奋"

公式C: [搞钱/赚钱] + [干货] + [时间承诺]
  → "搞钱干货：如何在60天无成本赚到第一桶金"
  → "真心建议：利用周末去培养成长型爱好"

公式D: [编年史/密码] + [人物]
  → "拼多多创始人黄峥「编年史」"
  → "DeepSeek创始人梁文锋编年史"
```

### 内容结构模板
```
【开头】名人背景介绍 + 素材来源说明（建立权威）
    ↓
【主体】时间线/主题线拆解
    - 原文引用（加粗/标色）
    - 关键决策分析
    - 思维模型提炼
    ↓
【结尾】升华到普通人可学习点 + 引流钩子
```

---

## 3. 视觉规范

### 封面设计
- **配色**: 低饱和度背景（黑/深灰）+ 高亮文字（绿/红/橙）
- **排版**: 时间轴式结构，类似文献档案
- **字体**: 等宽字体模仿打字机/代码风格
- **素材**: 博客截图/演讲摘录 + 手写式批注
- **品牌标识**: "SI" logo 固定位置

### 信息层级
```
标题（大字号，醒目）
  ↓
日期/时间锚点（绿色标记）
  ↓
原文引用（红色标注重点）
  ↓
解读分析（正常文字）
  ↓
结论/方法论（黄色底纹）
```

### 图片风格
- 真实截图为主（博客、演讲PPT）
- 多色标注系统：绿色=日期/背景，红色=重点，黄色=结论
- 保持"考古感"和"知识密度感"

---

## 4. 变现路径

### 产品矩阵
```
免费层: 名人早期素材整理（引流）
    ↓
低价层: 资料合集/工具包（信任建立）
    ↓
中价层: 《灯塔·商业知识库》订阅（主力）
    ↓
高价层: 1v1咨询/企业培训（高端）
```

### 引流钩子
- "完整版见《灯塔·商业知识库》"
- "评论区扣1领取大佬语录合集"
- "主页有惊喜"

### 评论区运营
- 置顶评论直接带货
- 高频回复建立专业感
- 引导私信深度交流

---

## 5. 数据指标

### 爆款标准
- 点赞 > 1000
- 收藏 > 点赞（说明干货属性）
- 评论 > 20（讨论质量）

### 内容保鲜期
- **长期价值**: 经典内容可持续6-12个月
- **追热点**: 新人物/新趋势及时跟进

---

## 6. 禁忌清单

- ❌ 不要搬运二手观点
- ❌ 不要过度承诺（少用"一定成功"）
- ❌ 不要视觉花哨（保持档案感）
- ❌ 不要忽视评论区运营

---

## 7. 执行检查清单

发布前检查:
- [ ] 素材来源是否可靠？
- [ ] 时间锚点是否清晰？
- [ ] 是否有1-3个可复用方法论？
- [ ] 引流钩子是否自然？
- [ ] 视觉是否统一（SI标识/配色）？

---

**记住**: 我们的价值是"时间差信息差"——帮用户省去看100篇博客的时间，直接给精华。
