# GLIMMER + 纳瓦尔认知成长模式
## 当前使用的完整笔记制作模式

---

## 📌 模式概述

**模式名称**: GLIMMER视觉锤 + 纳瓦尔长文结构  
**适用类型**: 认知成长/知识付费/深度干货  
**核心特征**: 高辨识度视觉 + 结构化深度内容

**设计理念**:
- **GLIMMER**: 荧光黄视觉锤，一眼识别，打破审美疲劳
- **纳瓦尔**: 7页长文深度，建立专业信任，高收藏率

---

## 🎨 视觉规范（GLIMMER风格）

### 封面标准

```bash
# 制作命令
python3 scripts/cover/run.py \
  --type glimmer \
  --title "主标题（10字内）" \
  --subtitle "副标题（15字内）" \
  --output "cover.png"
```

### 设计规格

| 元素 | 规格 | 说明 |
|------|------|------|
| **背景色** | RGB(225, 255, 0) 荧光黄 | 高饱和度，抢夺注意力 |
| **标题字体** | 梦源黑体/思源黑体 120pt | 黑色粗体，中文支持 |
| **副标题字体** | 梦源黑体/思源黑体 60pt | 灰色，次于主标题 |
| **尺寸** | 1080×1440px (3:4竖屏) | 小红书展示面积最大 |
| **页码** | **单图不标，多图才标** | 关键规则！ |
| **品牌栏** | 底部黑色条150px | "MODEL-SI 知识库" + "认知升级" |

### 封面公式: F = C + T + E
- **C (Contrast)**: 对比度>50%，前背景明度差
- **T (Typography)**: 标题8-15字，关键词放大1.5-2倍
- **E (Emotion)**: 情感传递 - 专业/温暖/真实

### 5种高点击率封面类型（可选变体）
1. **对比冲击型** - 前后对比图+大标题（+52%点击）
2. **痛点罗列型** - 纯色背景+数字标题（+40%点击）
3. **干货合集型** - 九宫格拼图+总标题（+35%点击）
4. **场景代入型** - 真人出镜+场景化标题（+45%点击）
5. **悬念互动型** - 问题标题+局部特写（+38%点击）

### 配色方案

```
主色：RGB(225, 255, 0) 荧光黄
辅色：RGB(0, 0, 0) 纯黑（标题）
辅色：RGB(80, 80, 80) 深灰（副标题）
品牌：RGB(0, 0, 0) 黑色底 + RGB(255, 255, 255) 白字
```

## 📝 文本规范（纳瓦尔式7页结构）

### 整体结构

```
Page 1: 封面页（GLIMMER视觉锤）
Page 2: 开篇故事（痛点/经历/钩子）
Page 3: 核心观点（反常识/金句爆破）
Page 4-6: 方法论展开（3-5个技巧+案例）
Page 7: 总结升华 + 行动引导（福利/互动）
```

### 详细结构

#### Page 1: 封面
```
[GLIMMER风格封面图]
主标题：数字+痛点+承诺
副标题：一句话解释价值
```

#### Page 2: 开篇故事
```
个人经历/用户痛点场景（2-3句）

示例：
"用了半年AI绘画工具，
今天给大家做个全面对比。

很多粉丝问我：
'到底该用哪个AI绘画工具？'"
```

#### Page 3: 核心观点
```
反常识观点/核心金句（加粗）

示例：
"其实每个工具都有优缺点，
关键是看你的需求。"

━━━━━━━━━━━━━━━
```

#### Page 4-6: 方法论（3-5个技巧）
```
🎯 技巧1：名称

优点：
• 要点1
• 要点2
• 要点3

缺点：
• 要点1
• 要点2

适合：人群描述

推荐度：⭐⭐⭐⭐⭐

━━━━━━━━━━━━━━━
```

#### Page 7: 总结 + 行动引导
```
💡 我的建议

人群1 → 推荐工具1（原因）
人群2 → 推荐工具2（原因）
人群3 → 推荐工具3（原因）

核心心法：
一句话金句总结

━━━━━━━━━━━━━━━

🎁 福利

整理好的资料包：
• 项目1
• 项目2
• 项目3

评论区扣"1"领取 👇
```

### 文案风格规范

| 元素 | 规范 |
|------|------|
| **语气** | 专业亲和，像朋友对话 |
| **人称** | 多用"你"拉近距离 |
| **句式** | 短句为主，易于阅读 |
| **Emoji** | 适度使用，增强可读性 |
| **分隔线** | `━━━━━━━━━━━━━━━` 区分章节 |
| **重点** | 关键信息加粗/变色 |

## 🔄 完整制作流程

### 阶段1: 选题（30分钟）
- 从选题库选择（认知升级/思维模型/个人成长/读书方法）
- 验证可行性（小红书搜索竞争情况）
- 确定使用的正文模板类型

### 阶段2: 调研（45-60分钟）
```bash
python3 scripts/search/run.py --keyword "关键词"
```
- 百度搜索截图
- 收集金句、数据、案例
- 准备1-2个个人案例

### 阶段3: 大纲（20分钟）
- 使用纳瓦尔7页结构规划
- 确定每页核心内容
- 选择正文模板类型

### 阶段4: 文案（60-90分钟）
- 标题：使用爆款标题公式，写3-5个备选
- 正文：按选定模板撰写
- 结尾：金句收尾 + 互动引导

### 阶段5: Canvas封面（15分钟）
```bash
python3 scripts/cover/run.py \
  --type glimmer \
  --title "标题" \
  --subtitle "副标题" \
  --output "cover.png"
```

### 阶段6: 视觉检查（5分钟）
- **必须用image工具查看封面**
- 确认中文正常显示
- 确认无"1/7"页码（单图情况）

### 阶段7: 发布（10分钟）
```bash
python3 scripts/publish/run.py --input draft.md
```

### 阶段8: 复盘（24-48小时后）
- 记录曝光/点赞/收藏/评论
- 分析爆款/低表现原因
- 更新选题库

## ❌ 本模式绝对禁止

| 禁止项 | 后果 | 正确做法 |
|--------|------|----------|
| 单图封面标"1/7" | 欺骗用户 | **单图不标页码** |
| 随机网络图片 | 与内容无关 | **强制Canvas制作** |
| 不检查视觉 | 中文方框/错误 | **image工具确认** |
| 偏离纳瓦尔结构 | 内容散乱 | **遵循7页结构** |
