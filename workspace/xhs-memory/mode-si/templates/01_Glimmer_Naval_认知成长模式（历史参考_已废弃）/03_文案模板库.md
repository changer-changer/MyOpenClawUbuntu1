# GLIMMER + 纳瓦尔模式 - 文案模板库
## 8种可套用的正文模板

---

## 模板1: 痛点解决型

### 适用场景
用户有明确痛点，提供解决方案

### 结构
```
【痛点共鸣】你是不是也...
【问题分析】其实这是因为...
【解决方案】分享3个方法：
  1. 方法一 + 具体步骤 + 案例
  2. 方法二 + 具体步骤 + 案例
  3. 方法三 + 具体步骤 + 案例
【效果展示】这样做之后...
【行动引导】建议收藏/评论区交流
```

### 示例标题
"为什么越学越焦虑？认知科学家揭示的真相"
"30岁还迷茫？用这3个思维模型找到方向"

### 使用要点
- 痛点要精准，直击用户内心
- 方法要具体，可操作性强
- 案例要真实，增强可信度

## 模板2: 数字清单型

### 适用场景
干货汇总，收藏率高

### 结构
```
【价值预告】读完...我总结出...
【清单内容】
  1. 要点一 + 解释 + 案例
  2. 要点二 + 解释 + 案例
  3. 要点三 + 解释 + 案例
  ...
【总结升华】核心观点
【行动引导】收藏/关注/私信
```

### 示例标题
"读完100本书后，我提炼出这7个认知升级心法"
"2024年最值得用的5个AI工具"

### 使用要点
- 数字要具体，降低决策成本
- 要点要精炼，一句话说明白
- 解释要深入，不只是罗列

---

## 模板3: 反常识观点型

### 适用场景
引发好奇，互动率高

### 结构
```
【反常识观点】直接抛出惊人观点
【解释原因】为什么会有这个结论
【科学依据】用研究/数据支撑
【实际案例】举例说明
【反思引导】引发用户思考
【互动邀请】评论区讨论
```

### 示例标题
"为什么越努力越焦虑？神经科学揭示的真相"
"直觉比理性更可靠？心理学家的反常识发现"

### 使用要点
- 观点要足够反常识，打破认知
- 论证要充分，不能无理取闹
- 引导要开放，邀请用户思辨

---

## 模板4: 故事启发型

### 适用场景
人设塑造，情感共鸣

### 结构
```
【故事开头】设置场景，引入人物
【冲突出现】遇到问题/挑战
【转折过程】如何思考和行动
【成长结果】获得什么改变
【启示总结】提炼核心观点
【互动引导】邀请用户分享
```

### 示例标题
"30岁裸辞后，我用3个月找到了人生方向"
"从月薪5千到年薪50万，我做对了什么？"

### 使用要点
- 故事要真实，细节要具体
- 冲突要清晰，转折要自然
- 启示要明确，与读者相关

---

## 模板5: 产品转化型

### 适用场景
知识付费产品推广

### 结构
```
【痛点共鸣】描述目标用户的困扰
【问题分析】分析原因，建立信任
【解决方案】引出产品
【产品介绍】详细说明内容和价值
【案例证明】用户成功案例
【限时优惠】制造紧迫感
【行动引导】立即购买/私信咨询
```

### 示例标题
"读完100本书，我把它熬成了一碗'认知高汤'"
"3年读了100本书，浓缩成这个专栏"

### 使用要点
- 痛点要精准，用户有共鸣
- 价值要清晰，解决什么问题
- 案例要真实，增强说服力

## 模板6: 热点结合型

### 适用场景
借势热点，快速获取流量

### 结构
```
【热点引入】简述热点事件
【观点连接】找到与认知成长的结合点
【深度分析】用认知框架解读热点
【方法论】提炼可学习的思维方法
【互动引导】邀请用户发表看法
```

### 示例标题
"从董宇辉事件看：普通人如何打造个人IP"
"ChatGPT爆火背后：AI时代的认知升级"

### 使用要点
- 热点要相关，不能硬蹭
- 分析要深度，不只说表面
- 方法要实用，读者能学到

## 模板7: 对比反差型

### 适用场景
制造冲突，吸引眼球

### 结构
```
【对比设置】A vs B
【A的问题】描述A的困境
【B的成功】描述B的成功
【原因分析】为什么差距这么大
【方法论】如何像B一样
【行动引导】立即行动
```

### 示例标题
"同样的起点，为什么5年后差距这么大？"
"A和B同时入职，5年后薪资差10倍"

### 使用要点
- 对比要鲜明，差距要明显
- 分析要客观，不能贬低一方
- 方法要可行，读者能模仿

---

## 模板8: 提问互动型

### 适用场景
提升评论互动率

### 结构
```
【问题抛出】提出一个引发思考的问题
【观点分享】分享你的看法
【案例支撑】用案例说明
【反向思考】提出另一个角度
【互动邀请】邀请用户发表看法
```

### 示例标题
"30岁，应该选择稳定还是冒险？"
"你支持'996'还是'工作生活平衡'？"

### 使用要点
- 问题要开放，没有标准答案
- 观点要鲜明，但不能绝对
- 引导要友好，欢迎不同声音

## 组合使用技巧

实际创作中可以**组合使用**多个模板：

**示例：数字清单 + 痛点解决**
```
标题：5个方法帮你告别学习焦虑

结构：
1. 痛点共鸣（你为什么焦虑）
2. 数字清单（5个方法）
   - 每个方法用"痛点解决"结构展开
3. 总结升华
```

**示例：故事启发 + 产品转化**
```
标题：从负债到年入百万，我做对了这3件事

结构：
1. 故事启发（我的逆袭经历）
2. 方法论（3个关键动作）
3. 产品转化（详细拆解在专栏）
```

## 使用建议

1. **根据选题选模板** - 不是每个笔记都用同一种
2. **灵活组合** - 可以2-3个模板组合使用
3. **保持结构清晰** - 让读者一眼看懂逻辑
4. **加入个人特色** - 模板是框架，内容是灵魂
5. **持续优化** - 根据数据反馈调整模板使用
