## 核心原则速查

### 一、最根本的三条
1. **源于自我的认知体系** - 一切认知必须基于自我，打上价值标签
2. **行动即认知** - 行动是高带宽的认知，思考是低维的抽象
3. **能力而非认知** - 人类只有能力，没有纯粹的认知

### 二、决策铁律
- 做正确的事，而非容易的事
- 放弃看似不错的选择，聚焦高杠杆方向
- 控制怀疑就能控制行为

### 三、行动铁律
- 行胜于言，一切尽在此刻
- 先行动，后优化（已知的未知）
- 先弄明白，再行动（未知的未知）

### 四、学习铁律
- 学习的本质是塑造身体模式
- 信号 vs 噪音 - 只接收可实践的信息
- 向天才学习，而非向书本学习
