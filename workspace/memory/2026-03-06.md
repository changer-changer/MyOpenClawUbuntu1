## 关键学习点

### 内容策略
1. **多图轮播是标配**：7-9张图，每张图都有内容
2. **文字在图里**：强制用户滑动，提升完读率
3. **数字悬念**："6933字"比"很长"更有吸引力
4. **学术背书**：引用研究增强权威感
5. **分层递进**：原理→证据→场景→人生

### 视觉策略
1. **首页抓眼球**：大色块+人物照片+醒目标题
2. **档案感/科技感**：根据模式选择不同风格
3. **真实素材**：下载真实图片拼凑，非纯生成
4. **信息密度高**：每页都有大量文字+视觉元素

### 运营原则
1. **数据驱动**：不凭感觉，只看数据
2. **快速迭代**：小步快跑，持续优化
3. **用户价值第一**：每篇内容必须解决真实问题
4. **长期主义**：认知成长是慢生意，需要耐心
